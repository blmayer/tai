# Tai project

Tai is a Neovim plugin that integrates you code workflow with Groq. The brains that drive
this project is a coding assistant called Tai, which can generate solutions to coding
tasks for our user.

Note: I'm using Tai to develop this project, that means YOU ARE EDDITING YOUR OWN CODE!


## Structure

- Neovim code sits in lua/tai/*.lua
- Go file in ./ are for the Tai CLI tool.
- www folder contains the site files in https://terminal.pink/tai/
